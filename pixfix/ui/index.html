Cleans up AI-generated pixel art — snap to grid, remove AA fuzz, reduce to a palette, remove backgrounds.
Quick Start
1. Open an image — press o or drag & drop
2. Preview the result — original and processed side by side
3. Tweak settings — switch to the Settings tab
4. Save — press s to export
Tabs
• Preview — original vs processed side by side
• Settings — tune all pipeline parameters
• Diagnostics — grid scores and color histogram
• Batch — process multiple images at once
• Sheet — split sprite sheets, normalize, reassemble

What you can do
• Snap to grid — auto-detects the pixel grid and aligns everything to it. Override the size and phase if needed.
• Downscale — reduce to logical pixel resolution using center-weighted, majority-vote, or center-pixel sampling.
• Remove anti-aliasing — clean up the soft blending that AI generators add between colors.
• Reduce colors — snap to built-in palettes, any of 3000+ Lospec palettes, a .hex file, or auto-extract the best N colors.
• Remove backgrounds — auto-detect the background color and make it transparent. Supports flood fill from edges or global removal.
• Resize output — upscale sprites for game engines (2x, 3x…) or set exact dimensions.
• Batch process — run the same pipeline on a folder of images at once.
• Sprite sheets — split a sheet into tiles (fixed grid or auto-detect from a white background), normalize each tile, reassemble, and generate animated GIFs.
